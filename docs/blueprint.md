# **App Name**: PillCheck AI

## Core Features:

- User Authentication: Allow users to sign in using Google account.
- User Profile: Collect and store user age and gender temporarily for session personalization.
- Manual Medicine Input: Allow users to manually enter medicine names to check for interactions and side effects.
- Image-Based Medicine Input: Allow users to upload medicine box images and extract medicine names using OCR. Use a tool to identify the medicines that appear in the images.
- Drug Interaction Check: Check for drug-to-drug interactions based on the entered medicines.
- Side Effect Check: Identify potential side effects associated with the entered medicines.
- AI Assistant (Vinnu): Implement an AI chatbot named 'Vinnu' to answer user questions about medicines, side effects, emergency steps, and precautions.
- Emergency Protocol: If side effects are reported, immediately display first-aid suggestions and emergency helpline numbers.
- Language Support: Implement a language selector with support for multiple Indian languages.

## Style Guidelines:

- Background color: White (#FFFFFF) for a clean and professional look.
- Primary color: Blue (#3498db) for informational elements and accents.
- Accent color: Green (#2ecc71) for positive feedback and confirmations.
- Warning color: Red (#e74c3c) for warnings, errors, and emergencies.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern feel suitable for a medical context.
- Use clear, understandable icons for different medicine types, symptoms, and emergency actions.
- Implement a mobile-first, card-based layout for clear organization and accessibility.
